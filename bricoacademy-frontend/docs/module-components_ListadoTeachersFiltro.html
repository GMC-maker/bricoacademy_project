Componente `ListadoTeachersFiltro`. Muestra un listado de profesores con opciones de filtrado por "activo" y varias opciones de exportación/impresión: - Captura en PDF mediante `html2canvas` + `jspdf` (botón "PDF(captura)"). - Generación de un PDF con `@react-pdf/renderer` usando `TeachersReportPdf`. Estado principal: - `datos`: array con los profesores obtenidos desde la API `/teachers/`. - `filtroActivo`: string con valores `""|"1"|"0"` para todos/activos/inactivos.
